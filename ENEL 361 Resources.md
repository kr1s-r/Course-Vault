# Course Content
Course Playlist: [ENEL 361 - Course Content](https://www.youtube.com/playlist?list=PL4DiSQXDsdMoHoxYKMe6rf7cARAeMnnoR)
### Course Overview
- This course focuses on diodes, the properties of semi-conductors and how to make a diode using pn-junctions. In addition, this course covers an introduction to transistors such as BJTs and MOSFETs.
