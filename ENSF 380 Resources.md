# ENSF 380/409 Resources

### Course Overview
- This course focuses on object-oriented principles (OOP) in Java. You will learn about classes, objects, and the four cornerstones of OOP: encapsulation, abstraction, inheritance, and polymorphism. In addition, you will learn about UML Diagrams, SOLID principles, and unit testing using JUnit.
